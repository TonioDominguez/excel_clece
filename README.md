# Curso de Microsoft Excel
 
Material del curso de iniciación a Microsoft Excel: dos clases de 6 horas, con diapositivas de apoyo y un libro de ejercicios para los alumnos.
 
## Contenido del repositorio
 
| Archivo | Descripción |
|---|---|
| `Clase1_Fundamentos_Excel.pptx` | Diapositivas de la Clase 1 — Fundamentos (apartados 1 a 12: entorno de Excel, celdas, gestión de archivos, formato, fórmulas y funciones básicas). |
| `Clase2_Analisis_Excel.pptx` | Diapositivas de la Clase 2 — Análisis de datos (apartados 13 a 19: tablas y gráficos, referencias, función SI, formato condicional, tablas dinámicas y funciones de texto). |
| `Libro_Maestro_Practicas_Excel.xlsx` | Libro de Excel con los ejercicios prácticos que se resuelven durante las dos clases. |
 
## Cómo usar este material
 
1. Descarga los tres archivos del repositorio.
2. Abre las diapositivas de cada clase para seguir el guion de la sesión (conceptos, ejemplos resueltos y ejercicios).
3. Usa `Ejercicios_Excel.xlsx` como punto de partida para que los alumnos resuelvan los ejercicios propuestos en cada apartado.
